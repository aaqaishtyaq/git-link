# git-link(1)

Generate Permalink for Github without leaving the terminal.

## Usage

```console
% git link cmd/root.go:1..15
https://github.com/aaqaishtyaq/git-link/blob/71307f9fabfa31266c9945d696c3e21cd332b021/cmd/root.go#L1-L15
```
