# git-link(1)

Generate Permalink for Github without leaving the terminal.

## Usage

```console
% git link cmd/root.go:1..5
https://github.com/aaqaishtyaq/git-link/blob/c710261ce7bc4c96dd0365bf8d570ed0cc254c0c/cmd/root.go#L1-L5
```
