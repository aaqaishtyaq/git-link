# git-link(1)

Generate Permalink for Github without leaving the terminal.

## Usage

```console
% git link cmd/root.go:1..5
https://github.com/aaqaishtyaq/git-link/blob/2bb2e57bbb24b7b7e1ca2bf09b25dd701e7099f2/cmd/root.go1..5
```
